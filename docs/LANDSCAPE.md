## 1. Hard constraints any proof must respect

**Undecidability of the generalized class.** Conway proved that for any computable f there is a map g with g(n)/n periodic rational such that 2^f(n) = g^k(2^n), hence no algorithm decides, given such a g and n, whether some iterate g^k(n)=1 — and this holds even in the purely multiplicative case b_i = 0 (Conway 1972, https://gwern.net/doc/cs/computable/1972-conway.pdf). Conway's own caveat is the load-bearing constraint: "our theorem says nothing about the Collatz game," but it prohibits any *uniform* solution to games of this type. Any successful proof must exploit arithmetic specific to (3x+1)/2, not generic orbit machinery.

**Π⁰₂-completeness, sharpened.** Kurtz–Simon strengthened Conway from powers-of-2 starting points to all starting points: the set of generalized Collatz functions g with range-g = ω (every integer reaches 1) is Π⁰₂-complete, via a double register-machine simulation converting totality into all-orbits-reach-1 (Kurtz–Simon 2007, https://people.cs.uchicago.edu/~simon/RES/collatz.pdf). So the generalized problem sits at the exact top of what a single proof template could decide; Π⁰₂ is optimal.

**Arithmetic-hierarchy asymmetry dictates strategy.** The full conjecture (∀n ∃k T^k(n)=1) is Π⁰₂. Its two failure modes split: "no nontrivial cycles" is Π⁰₁ (a cycle is a finite verifiable witness), so independence from a sound theory would imply its *truth*; "no divergent orbits" is genuinely Π⁰₂ — a divergent orbit is an infinite object, and "if a Collatzian game does not stop, then there is no proof of this" in general (Conway 2013, https://raganwald.com/assets/fractran/Conway-On-Unsettleable-Arithmetical-Problems.pdf). Consequently: independence results can never settle the full conjecture, and a disproof-by-divergence could itself be unprovable. No Π⁰₁-equivalent form of the full conjecture (e.g., a computable excursion bound) is known; producing one would be a structural breakthrough.

**Unsettleability is a live possibility, not a rhetorical flourish.** Conway exhibited an explicit 24-fraction FRACTRAN game that never stops but unprovably so (improved to 7 fractions by Rickard), and his mountain/CRT postscript argues each googol-scale "mountain" has positive probability of closing into a cycle, making a proof of "no large cycles" extremely unlikely and Collatz itself "very likely to be unsettleable" (Conway 2013, same URL). His Appendix 1 concedes exactly one non-inconceivable proof route: a circle-method-style argument converting the drift heuristic into a theorem.

**Why soft/measure arguments provably stall.** Tao's Remark 1.4: if any single orbit never dropped below a constant C₀, probabilistic heuristics predict a *positive-density* set of seeds would be absorbed by it; so almost-all methods structurally cannot exclude even one bad orbit, and replacing f(N)→∞ by an absolute constant "is likely to be almost as hard to settle as the full Collatz conjecture" (Tao 2019, https://arxiv.org/abs/1909.03562). Naive iteration of Terras/Korec fails because the logarithmic/uniform measure is not invariant under T — iteration can dump almost all of [1,x] into the density-zero exceptional set at scale x^θ (Tao 2019, §1.1).

**Ergodic theory sees only noise.** T on ℤ₂ is exactly conjugate to the full one-sided 2-shift via the parity-vector map Q_∞, and is strongly mixing for Haar measure (Lagarias 1985, https://www.cecm.sfu.ca/organics/papers/lagarias/paper/html/paper.html; Matthews–Watts 1984). The conjecture is equivalent to ℤ⁺ ⊆ Φ((1/3)ℤ) — a statement about where the Haar-measure-zero set of integers sits inside ℤ₂ (Bernstein 1994, https://www.ams.org/journals/proc/1994-121-02/S0002-9939-1994-1186982-9/S0002-9939-1994-1186982-9.pdf). Any proof must exploit structure invisible to both Haar measure and finite automata.

**No-go theorems for automated methods.** Zantema's 7-rule unary rewriting system (termination ⟺ Collatz) admits *no* termination proof via natural matrix interpretations of any dimension, even with dependency pairs (Yolcu–Aaronson–Heule 2021, https://arxiv.org/abs/2105.14697). And no infinite semilinear/Presburger-definable T-invariant set avoiding {1,2} exists (corollary of AP-sufficiency: Monks 2006, https://monks.scranton.edu/files/pubs/SufficiencyRev4.pdf), so any "regular divergence certificate" must be sparse and non-semilinear — or finite, i.e., exactly a nontrivial cycle.

**Where the practical frontier sits.** The bbchallenge cryptids (Antihydra, Hydra, Bigfoot) show the entire modern decider arsenal fails precisely on statements about the parity statistics of individual orbits of n ↦ ⌊3n/2⌋ — equivalently the parity of ⌊K(3/2)^n⌋, the genre of Mahler's Z-number problem, open since 1968 (https://wiki.bbchallenge.org/wiki/Antihydra). Individual-orbit parity control is the named barrier.

## 3. The Busy Beaver connection, made precise

### 3.1 Collatz-like iteration is the generic mechanism of extremal small programs

Michel's dictionary: f is a *generalized Collatz mapping* iff ∃d ≥ 2, integers a_i, b_i with f(dn+i) = a_i n + b_i; *pure Collatz-like* = partial (halting residues); *exponential Collatz-like* = f(dn+i) = (a_i p^n + b_i)/c_i (Michel 2015, https://arxiv.org/abs/1311.1029). The BB(5) champion iterates g(x) = (5x+18)/3 [x≡0 mod 3], (5x+22)/3 [x≡1], ⊥ [x≡2] from x=0; the orbit 0→6→16→…→12284→⊥ (15 steps, each ~(5/9)x² TM steps) *is* the 47,176,870-step runtime (Aaronson 2020, https://www.scottaaronson.com/papers/bb.pdf, after Michel https://arxiv.org/abs/0906.3749). "Most other strong contenders over the past 30 years have also applied Collatz-like maps… this raises the possibility—whether enticing or terrifying!—that progress on determining the small values of BB(n) might go hand in hand with progress on the Collatz Conjecture" (Aaronson 2020, p. 11). Michel proved the 2-, 3-, 4-state Busy Beavers halt on *every* input, and the BB(5) champion halts on every input iff **Conjecture 22**: g reaches ⊥ from every natural start — still open, a minimal Collatz-sibling produced by extremal computation rather than human choice (Michel 1993; Aaronson 2020 §5.6).

### 3.2 The cryptid frontier: exactly where certificates fail

- **Antihydra** (BB(6) gatekeeper, mxdys 2024): iterate H(n)=⌊3n/2⌋ from 8, counter c += 2 on even, −1 on odd; halts iff c hits −1. Equivalent to the parity distribution of ⌊K(3/2)^n⌋ (via D(n+1)=⌈3D(n)/2⌉ and Odlyzko–Wilf) — Mahler's-3/2-problem genre. Residual halting probability < ((√5−1)/2)^(2^37) ≈ 10^{−2.87×10^10} after 2^38 simulated steps, yet no proof (https://wiki.bbchallenge.org/wiki/Antihydra; Ligocki 2024, https://www.sligocki.com/2024/07/06/bb-6-2-is-hard.html).
- **Hydra** (BB(2,5)): same map from 3 with mirrored termination (Ligocki 2024, https://www.sligocki.com/2024/05/10/bb-2-5-is-hard.html). **Bigfoot** (BB(3,3)): b ↦ (4/3)b + c on residues mod 6, halting requires b≡2 (mod 6) with a=0 where a random-walks with drift +1/3 (Ligocki 2023, https://www.sligocki.com/2023/10/16/bb-3-3-is-hard.html). Bigfoot compiles to 7 states 2 symbols, Hydra to 9 — so BB(7) is Collatz-hard too (https://wiki.bbchallenge.org/wiki/Cryptids).
- **Space Needle** (BB(6), 2025): halts iff b ↦ b + v(b) + (3/2)(b/2^{v(b)} − 1) from 6 hits a power of 2 — a 2-adic valuation-of-sums problem, different in kind from drift cryptids (https://wiki.bbchallenge.org/wiki/Space_Needle).
- BB(6) status (June 2026): ~1094 holdouts, all simulated to 10¹³ steps; mxdys's new FAR variant decided 113/1534 on first application (https://wiki.bbchallenge.org/wiki/BB(6); Brubaker/Quanta 2025, https://www.quantamagazine.org/busy-beaver-hunters-reach-numbers-that-overwhelm-ordinary-math-20250822/).

### 3.3 Certificate technologies and their Collatz analogues

The BB(5) pipeline architecture — untrusted, SAT/heuristic *search* emits small certificates; a proved-correct *verifier* checks them inside the kernel (only 40 FAR/WFAR certificates are hardcoded in Coq; the search ran outside) — is the template to port (bbchallenge 2025, https://arxiv.org/abs/2509.12337, §4.2).

| BB technology | Mechanism | Collatz-space analogue |
|---|---|---|
| Loops / (translated) cyclers (175.4M machines) | Record-breaking configuration recurrence | Cycle detection; trivial for Collatz (cycle = counterexample), useful for Collatz-like map surveys |
| CTL / regular CTL: NGramCPS (6.0M), RepWL (6,577) | Regular language C of configurations: contains start, step-closed, no halting config — an inductive invariant | A nonempty 2-automatic T-invariant set avoiding {1,2} would be a *divergence certificate* (open both directions, §4); dually, regular over-approximations of Pred(1) on binary/ternary digit strings for *convergence* of restricted fragments |
| FAR (23 machines): NFA over tape-words, SAT-searched; verification = 8 matrix inequalities | co-CTL: NFA accepts all eventually-halting configs, rejects start | SAT-search NFAs over mixed binary-ternary Collatz representations (the Yolcu–Aaronson–Heule alphabet {f,t,0,1,2,/,.} is exactly the right carrier, https://arxiv.org/abs/2105.14697); verifier is ~100 lines of linear algebra, ideal for Lean |
| WFAR (17 machines): deterministic weighted automata, integer weights, Bellman–Ford feasibility | Non-regular invariants; weights capture counter/drift structure (0^n1^n-type) | Weighted-automaton invariants whose weight tracks ν₂/ν₃ valuations or odd/even counters of Collatz-like orbits — precisely the +2/−1 structure of Antihydra/Hydra; the known barrier is that no certificate class yet expresses a *positive-drift* argument |
| Bouncers: inductive shift rules B_n → B_{n+1}, formally verified (busycoq/Bouncers.v) | Parametrized acceleration lemmas config(a,b) →* config(f(a,b)) | Multi-step Collatz acceleration lemmas, e.g. H^s(2^s t)=3^s t, H^s(2^s t+1)=3^s t+1 (Kuperberg's near-linear Antihydra simulator; https://wiki.bbchallenge.org/wiki/Hydra_function) — digit-block abstraction linearizes 3/2-type maps |
| Sporadic hand proofs (13 machines, e.g. Skelet #17 ≈ 7,000 Coq lines; Xu 2024, https://arxiv.org/abs/2407.02426) | Individual invariant constructions | Template for one-off proofs about specific Collatz-like maps (e.g., Conjecture 22) |

**The two-sided lesson.** Positive: the 2024 Coq-BB5 result proves that individual Collatz-like non-halting statements of nontrivial difficulty *can* be settled and formally verified by decider technology — the 25 pre-2020 holdouts Aaronson asked about all fell (Aaronson 2024, https://scottaaronson.blog/?p=8088). Negative: the arsenal fails exactly on drift cryptids, i.e., on individual-orbit parity statistics — which is the *divergence* side of Collatz. Judgment: port certificate technology to the cycle/invariant-set side of Collatz and to the decidability-boundary mapping of Michel's class; do not expect it to touch divergence without a genuinely new certificate class that internalizes drift.

## 4. Automata-theoretic and p-adic structure

**The finite-horizon layer is completely tame.** Stopping-time classes are congruence classes mod 2^k (Terras 1976); each bounded-odd-steps layer S_i = {k : exactly i odd steps to 1} is 2-automatic (Shallit–Wilson 1992, https://cs.uwaterloo.ca/~shallit/Papers/wilson.pdf), with explicit regexps exponential in k (Stérin 2020, https://arxiv.org/abs/1907.00775, generalizing Colussi 2011). The one-step graph of T is Presburger-definable, so any *proposed* regular certificate is decidably checkable (Büchi arithmetic closure properties, Bruyère–Hansel–Michaux–Villemaire 1994). What is open is precisely the unbounded closure: whether Pred(1) is 2-automatic, and whether a sparse non-semilinear 2-automatic T-invariant set avoiding {1,2} exists — the infinite semilinear case is dead (Monks 2006 corollary), the finite case is the cycles problem.

**Non-definability evidence.** The reachability relation of T_{q,d} (q+d a power of 2, includes 3x+1) is not definable in base-q Büchi arithmetic, via Cobham (Dhiman–Pandey 2026, https://arxiv.org/abs/2602.06066 — math.GM, unrefereed; base-2 explicitly left open). Base-2 definability of Collatz reachability is a concrete open target with a decidable payoff either way.

**2-adic rigidity.** Q_∞ conjugates T to the full 2-shift; Φ = Q_∞^{−1} has explicit series form, and Collatz ⟺ ℤ⁺ ⊆ Φ((1/3)ℤ) (Bernstein 1994, URL above). Φ mod 2^n is a permutation of order exactly 2^{n−4} (n ≥ 6) with conjecturally only odd fixed points −1, 1/3 (Bernstein–Lagarias 1996). Aut(T) = {id, Ω}, and Ω(ℚ)⊆ℚ would kill divergent integer orbits (Monks–Yazinski 2004, https://monks.scranton.edu/files/pubs/AutoConjV13.pdf). Lagarias's Periodicity Conjecture (Q_∞(ℚ_odd) ⊆ ℚ_odd) implies no divergent trajectories; the converse inclusion is a theorem (Lagarias 1985).

**3-adic mixing (Tao's engine).** The Syracuse random variables Syrac(ℤ/3^nℤ) = F_n(Geom(2)^n) — the stationary law of x ↦ (3x+1)/2^a w.p. 2^{−a} — mix at fine scales: |char. function| ≪_A n^{−A} for ξ ∤ 3, proved via a 2D renewal process avoiding well-separated "black triangles" (Tao 2019, §7). The entropy heuristic predicts exponential mixing (unproven). Tao's β=1 equidistribution conjecture (c_n = 3^{−βn+o(n)}, β ≥ 1 known) would give #{N ≤ x : Col_min(N)=1} ≫ x^{1−o(1)}, nearly closing preimage density (Tao 2020, https://terrytao.wordpress.com/2020/01/25/equidistribution-of-syracuse-random-variables-and-density-of-collatz-preimages/). Natural-density upgrade requires mixing of the full random affine map, called "plausibly attainable" and still unpublished (Tao 2019, Remark 1.16).

**Alternative carriers.** Collatz embeds base-3→base-2 conversion in a 2D quasi-CA; predicting half the bits of T^i(x), i = O(log x), is in NC¹∖AC⁰ (Stérin–Woods 2020, https://arxiv.org/abs/2007.06979). In the Akiyama–Frougny–Sakarovitch base-3/2 system, T is digit-*appending* (odd n: ⟨T(n)⟩ = ⟨n⟩1), but the expansion language is not context-free, so this route leaves regularity behind (Eliahou–Verger-Gaugry 2025, https://arxiv.org/abs/2504.13716). De Mol's 2-tag system {a→bc, b→a, c→aaa} is the minimal combinatorial presentation, sitting at the exact tag-system decidability frontier (De Mol 2008/2009, https://arxiv.org/abs/0906.3329).

**Stochastic calibration targets** (any candidate framework should reproduce these): mean stopping 6.95212 log n, Gaussian s.d. 10.07√(log n), extremal constant 41.677647 with exact forward/backward duality, excursion exponent 2; Brownian-with-drift log(3/4) structure theorem for (d,g,h)-maps (Kontorovich–Sinai 2002, https://arxiv.org/abs/math/0601622; Kontorovich–Lagarias 2009, https://arxiv.org/abs/0910.1944). The unproved model-mean lower bound marks the precise finite/infinite-horizon frontier.

## 5. Formalization landscape

**Lean/mathlib.** mathlib4 contains zero Collatz code (verified via GitHub search, 2026-07-20). The canonical open statement lives in google-deepmind/formal-conjectures: `collatzStep` + `collatz_conjecture` (also aliased as Erdős #1135), plus the BB(15)-equivalent Erdős base-3 statement `CollatzLike` (https://github.com/google-deepmind/formal-conjectures).

**ccchallenge.org** (Stérin/Rozier, launched ~Feb 2026): bbchallenge-style pipeline to formalize the Collatz literature, 371 papers tracked; as of 2026-07-20: 1 audited (Böhm–Sontacchi 1978 cycle equation, https://github.com/tcosmo/BohmSontacchi1978_lean), Tao 2019/2022 claimed sorry-free but **awaiting statement audit** (https://github.com/gotrevor/tao-collatz — axioms only propext/choice/Quot.sound; audit fidelity of the frozen Colmin/log-density definitions is the open risk), Terras 1976 in progress (https://github.com/lechmazur/terras_density_one), plus Bernstein–Lagarias 1996, Eliahou 1993 (via Harmonic Aristotle), Krasikov–Lagarias 2003, Monks 2006 in flight — all AI-assisted (https://ccchallenge.org/). Strategic implication: state any new result against ccchallenge's frozen definitions and submit there; do not formalize in isolation.

**Coq/Rocq.** Coq-BB5: S(5)=47,176,870 in 27,274 lines + 638 lemmas (+10,553 lines busycoq), single extra axiom, proof-by-reflection deciders, 45-min compile via native_compute (https://github.com/ccz181078/Coq-BB5; https://arxiv.org/abs/2509.12337). busycoq's untrusted-Rust-search/verified-checker split and its parametrized acceleration-lemma style are the reusable pattern (https://github.com/meithecatte/busycoq). FRACTRAN halting undecidability (Conway-style) is mechanized in coq-library-undecidability via TM→Minsky→FRACTRAN (Larchey-Wendling–Forster, https://arxiv.org/abs/2003.04604); Kurtz–Simon has never been mechanized anywhere. Partial BB(6) Rocq work: https://github.com/ccz181078/busycoq/tree/BB6.

**Isabelle.** AFP is a green field: no Collatz, FRACTRAN, or generalized-Collatz entry; nearest are Minsky_Machines and an abstract Busy_Beaver noncomputability entry (https://www.isa-afp.org/entries/index.json).

**Tooling judgments.** Kernel `decide` works for finite orbit checks only with structural/fuel recursion (well-founded recursion doesn't reduce); `native_decide` scales to millions of checks but trusts the compiler and is banned from mathlib — the busycoq-style certificate-checker design sidesteps the tradeoff entirely and is the right default. Caution list: johnjanik/syracuse-confinement (9 axioms, 12 sorries, load-bearing gap), collatz-lab.org, and the preprints.org Isabelle "proof" are unvetted; treat as noise.

**Realistically formalizable now:** (a) a mathlib base library (step/Syracuse maps, parity vectors, stopping time, Böhm–Sontacchi cycle equation — Terras's congruence structure is elementary); (b) Eliahou/Hercher-style cycle bounds *given* the linear-forms input as a clearly-flagged hypothesis (full Baker/Rhin bounds are a major separate project); (c) CTL/FAR/WFAR verifiers (small linear algebra); (d) Conway/Kurtz–Simon undecidability ported from the Coq FRACTRAN chain; (e) Antihydra-style orbit equivalences (basics already in Lean: https://github.com/rwst/Antihydra-Basics). *Not* realistically formalizable soon: Tao's full argument from scratch (hence the value of auditing the existing claimed formalization rather than rebuilding).

## 6. Ten most promising concrete next actions

1. **Build a Lean 4 CTL/FAR certificate verifier for Collatz-like maps on digit strings + SAT-driven certificate search** (port bbchallenge's search/verify split; carrier = Yolcu–Aaronson–Heule's mixed binary-ternary alphabet). — The single proven-successful industrial technology for Collatz-like nonhalting has never been pointed at Collatz-space itself (bbchallenge 2025, https://arxiv.org/abs/2509.12337; Yolcu et al. 2021, https://arxiv.org/abs/2105.14697).
2. **Re-run Hercher's continued-fraction bootstrap against X₀ = 2^71** (and current linear-forms constants), targeting m ≥ 92 → higher and K > 1.375×10¹¹ → higher. — 2^71 is 2.7× the bound Hercher needed and its consequences have not appeared in print; the computation is automatable (Hercher 2023, https://arxiv.org/abs/2201.00406; Barina 2025, https://pcbarina.fit.vutbr.cz/).
3. **Execute Ben Green's route: extract an explicit C_δ from Tao's argument and combine with verification ≤ 2^71 to prove Collatz on a set of positive logarithmic density.** — Explicitly flagged as in-principle possible (Tao 2019, Remark 1.4 fn. 3, https://arxiv.org/abs/1909.03562) and apparently never attempted; it would be the first unconditional "positive proportion reaches 1" statement.
4. **Compute Tao's c_n exactly via the Lemma 1.12 recursion to large n and test β = 1 numerically; then attack the natural-density upgrade (Remark 1.16).** — β=1 would yield x^{1−o(1)} preimage density, and Tao rates the natural-density upgrade "plausibly attainable" yet it remains unpublished (Tao 2020, https://terrytao.wordpress.com/2020/01/25/); cheap experiment, high-information.

# Lemma 0: the Berg–Meinardus equation as a Cartier–Mahler kernel, and a bi-Mahler exclusion for exotic Collatz components

2026-07-20. Status: Lemma 0 and the solution-space structure are elementary and
fully proved below (symbolically verified through z^60, see §6); the bi-Mahler
exclusion (Theorem 4.7) is, to our knowledge, unrecorded — it assembles
Adamczewski–Bell 2017, an elementary rationality argument, and Monks 2006, each
step proved or precisely cited. This note writes up properly the "v2 upgrade"
paragraph of `docs/notes/two-bases.md` and stakes the bridge described in
`docs/CRACKS.md`, Crack 1. Everything is a constraint on *disproofs* of Collatz
(certificate syntax), not progress on the conjecture itself.

## 1. Setup

**The map.** T : ℕ → ℕ, T(n) = n/2 for n even, T(n) = 3n+1 for n odd. The
Collatz conjecture: for every n ≥ 1 the forward orbit orb(n) := {T^j(n) : j ≥ 0}
contains 1. The orbit of 1 is the trivial cycle {1, 4, 2}.

**The graph.** G is the functional graph on vertex set ℕ with edge n → T(n) for
every n. A *weakly connected component* ("component") of G is an equivalence
class of the equivalence relation ~ generated by adjacency (n ~ m whenever
m = T(n) or n = T(m)).

**Lemma 1.1 (components = orbit-merging classes).** For a, b ∈ ℕ: a ~ b iff
orb(a) ∩ orb(b) ≠ ∅.

*Proof.* Write a ≐ b for orb(a) ∩ orb(b) ≠ ∅. ≐ is reflexive and symmetric.
Transitive: if u = T^i(b) ∈ orb(a) ∩ orb(b) and v = T^j(b) ∈ orb(b) ∩ orb(c),
then WLOG i ≤ j, so v = T^{j−i}(u) ∈ orb(u) ⊆ orb(a), hence v ∈ orb(a) ∩ orb(c).
Adjacent vertices satisfy ≐ (m = T(n) ⟹ m ∈ orb(n) ∩ orb(m)), and ~ is the
smallest equivalence relation containing adjacency, so ~ ⊆ ≐. Conversely if
T^i(a) = T^j(b) = q then a ~ T(a) ~ … ~ q and b ~ … ~ q, so a ~ b. ∎

**Lemma 1.2 (basic component facts).**
(i) Every component is forward-closed: n ∈ C ⟹ T(n) ∈ C.
(ii) {0} is a component of G.
(iii) Every component C ⊆ ℕ≥1 is infinite.

*Proof.* (i) n and T(n) are adjacent. (ii) T(0) = 0, and T(n) = 0 forces n = 0
(3n+1 ≥ 1; n/2 = 0 ⟹ n = 0), so 0 is adjacent only to itself. (iii) For n ≥ 1,
T(2n) = n, so 2n ~ n; iterating, {2^j n : j ≥ 0} ⊆ C. ∎

By (ii), the components of G are {0} together with the components of the
restriction of G to ℕ≥1. A component C ⊆ ℕ≥1 is **exotic** if 1 ∉ C. The Collatz
conjecture says exactly that no exotic component exists: by Lemma 1.1, n ~ 1 iff
orb(n) meets orb(1) = {1, 4, 2}, iff orb(n) ∋ 1 (from 4 or 2 the orbit continues
4 → 2 → 1).

**The Berg–Meinardus equation** [BM94, BM95]. Let D = {z ∈ ℂ : |z| < 1}, H(D)
the holomorphic functions on D, ω = e^{2πi/3}. The equation for h ∈ H(D):

    (BM)    h(z³) = h(z⁶) + (1/(3z)) · Σ_{j=0}^{2} ω^j h(ω^j z²).

All substitutions map D to D, so both sides are holomorphic on D ∖ {0}; the
root-of-unity average will turn out to vanish to order ≥ 4 at 0 (proof of
Lemma 2.1, Step 1), so the right side extends holomorphically to D and (BM) is
read as an identity in H(D).

**Cartier operators.** For h = Σ_{n≥0} a_n z^n define M_k h(z) = h(z^k) and the
section (Cartier-type) operators Λ_{k,r}(Σ a_n z^n) = Σ_{n≥0} a_{kn+r} z^n,
0 ≤ r < k. Write Λ₃ := Λ_{3,2}. All of these preserve H(D): if
limsup |a_n|^{1/n} ≤ 1 then limsup |a_{3n+2}|^{1/n} ≤ 1. Define

    𝔏 := 1 − M₂ − z·M₂·Λ₃

(the operator called L in `docs/CRACKS.md`; we reserve L for certificates).

## 2. Lemma 0 (normalization)

**Lemma 2.1.** Let h(z) = Σ_{n≥0} a_n z^n ∈ H(D). The following are equivalent:

  (a) h satisfies (BM);
  (b) 𝔏h = 0;
  (c) a_{2k} = a_k for all k ≥ 0, and a_{2k+1} = a_{3k+2} for all k ≥ 0;
  (d) a_n = a_{T(n)} for all n ≥ 1;
  (e) n ↦ a_n is constant on each weakly connected component of G.

*Proof.*

**Step 1: (a) ⟺ (c), by the coefficient computation.** All series below
converge absolutely and locally uniformly on D, and Taylor coefficients at 0
are unique, so (BM) holds iff the coefficient of z^m agrees on both sides for
every m ≥ 0. Expand:

    h(z³) = Σ_{n≥0} a_n z^{3n},        h(z⁶) = Σ_{n≥0} a_n z^{6n}.

For the averaged term, ω^j h(ω^j z²) = Σ_n a_n ω^{j(n+1)} z^{2n}, so

    Σ_{j=0}^{2} ω^j h(ω^j z²) = Σ_{n≥0} a_n ( Σ_{j=0}^{2} ω^{j(n+1)} ) z^{2n}
                              = 3 Σ_{n ≡ 2 (mod 3)} a_n z^{2n},

since Σ_{j=0}^{2} ω^{j(n+1)} = 3 if 3 | n+1 (i.e. n ≡ 2 mod 3) and = 0
otherwise (geometric sum: (ω^{3(n+1)} − 1)/(ω^{n+1} − 1) = 0 when ω^{n+1} ≠ 1).
The root-of-unity average thus extracts exactly the indices n ≡ 2 (mod 3);
writing n = 3k+2, the smallest exponent appearing is z⁴, so dividing by 3z is
legitimate in H(D) and

    (1/(3z)) Σ_{j=0}^{2} ω^j h(ω^j z²) = Σ_{k≥0} a_{3k+2} z^{6k+3}.

Hence (BM) reads

    Σ_{n≥0} a_n z^{3n}  =  Σ_{n≥0} a_n z^{6n}  +  Σ_{k≥0} a_{3k+2} z^{6k+3}.

Every exponent on either side is divisible by 3, so for m ≢ 0 (mod 3) the
comparison is vacuous (0 = 0). The left side contributes a_{m/3} at each
m ≡ 0 (mod 3); on the right, the first sum is supported on m ≡ 0 (mod 6) and
the second on m ≡ 3 (mod 6). Comparing:

  * m = 6k:   a_{2k} = a_k        (k ≥ 0);
  * m = 6k+3: a_{2k+1} = a_{3k+2} (k ≥ 0).

These are exactly the relations (c), and no others arise.

**Step 2: (b) ⟺ (c).** Coefficientwise, 𝔏h = h(z) − h(z²) − z·(Λ₃h)(z²) has
coefficient a_{2k} − a_k at z^{2k} (k ≥ 0) and a_{2k+1} − a_{3k+2} at z^{2k+1}
(k ≥ 0), since z·(Λ₃h)(z²) = Σ_k a_{3k+2} z^{2k+1}. So 𝔏h = 0 ⟺ (c).
Equivalently: applying the injective substitution M₃ to 𝔏h = 0 reproduces (BM)
term by term — M₃(𝔏h) = h(z³) − h(z⁶) − Σ_k a_{3k+2} z^{6k+3} — which is how
(BM) "cancels z ↦ z³" into the single mixed equation 𝔏h = 0 in the crossed
algebra generated by M₂ and Λ₃ over ℂ(z).

**Step 3: (c) ⟺ (d).** (c) ⟹ (d): for even n = 2k, k ≥ 1, the first relation is
literally a_n = a_{T(n)}. For odd n = 2k+1: T(n) = 6k+4, and the first relation
at index 3k+2 gives a_{6k+4} = a_{2(3k+2)} = a_{3k+2}, while the second gives
a_{2k+1} = a_{3k+2}; hence a_n = a_{T(n)}. (d) ⟹ (c): the first relation for
k ≥ 1 is a_{2k} = a_{T(2k)} (and is trivial for k = 0); the second follows from
a_{2k+1} = a_{T(2k+1)} = a_{6k+4} and a_{6k+4} = a_{T(6k+4)} = a_{3k+2}.

**Step 4: (d) ⟺ (e).** The constraint set {a_n = a_{T(n)} : n ≥ 1}, together
with the vacuous n = 0 instance, forces constancy exactly on the classes of the
equivalence relation generated by {n ~ T(n)}, i.e. (Lemma 1.1) on the weakly
connected components of G. Conversely constancy on components implies (d) since
n and T(n) share a component (Lemma 1.2(i)). ∎

Note what is *not* constrained: a_0 is tied to nothing (the k = 0 instance of
the first relation is a_0 = a_0), matching the fact that {0} is its own
component.

## 3. The solution space; Berg–Meinardus's theorem as a corollary

For a component C of G let h_C := Σ_{n∈C} z^n, the indicator series; h_C ∈ H(D)
since its coefficients are bounded. For C = {0}, h_C = 1; if the Collatz
conjecture holds, the unique component of ℕ≥1 has h_C = z/(1−z).

Let V ⊆ H(D) denote the set of solutions of (BM). By Lemma 2.1, h ∈ V iff its
coefficient sequence is constant on components; write c_C for the common value
on C, and m_C := min C.

**Proposition 3.1 (structure of V).**
(i) Each h ∈ V is determined by, and determines, its component values (c_C);
    coefficientwise h = c_{{0}}·1 + Σ_{C ⊆ ℕ≥1} c_C h_C.
(ii) A family of component values (c_C) arises from an element of V iff for
     every ε > 0, |c_C| ≤ (1+ε)^{m_C} for all but finitely many components C.
(iii) The solutions with bounded coefficient sequence are exactly the bounded
      families (c_C) ∈ ℓ^∞ — "bounded functions constant on the components".
(iv) The family {1} ∪ {h_C : C a component of ℕ≥1} is linearly independent over
     ℂ. Consequently, if G restricted to ℕ≥1 has κ components then
     dim_ℂ V = κ + 1 when κ < ∞, and V is infinite-dimensional when κ = ∞.

*Proof.* (i) is Lemma 2.1(e) (components partition ℕ, so the coefficientwise
sum is well-defined). (ii) (⇐): given ε > 0, all but finitely many C satisfy
|c_C| ≤ (1+ε)^{m_C} ≤ (1+ε)^n for every n ∈ C (as n ≥ m_C); the finitely many
exceptional components contribute coefficients bounded by some constant M, and
M ≤ (1+ε)^n for large n; hence limsup |a_n|^{1/n} ≤ 1+ε for every ε, so the
radius of convergence is ≥ 1 and h ∈ H(D). (⇒): if h ∈ H(D) then
limsup |a_n|^{1/n} ≤ 1 (radius ≥ 1 by holomorphy at 0 — Taylor series of h on D),
so for each ε there is N with |a_n| ≤ (1+ε)^n for n ≥ N; every component with
m_C ≥ N satisfies |c_C| = |a_{m_C}| ≤ (1+ε)^{m_C}, and only finitely many
components have m_C < N (their minima are distinct). (iii) is immediate.
(iv) In a finite vanishing linear combination λ·1 + Σ_i λ_i h_{C_i} = 0, the
coefficient at z^0 gives λ = 0 (each C_i ⊆ ℕ≥1 misses 0) and the coefficient at
z^{m_{C_i}} gives λ_i = 0 (components are disjoint). The dimension count
follows from (i), (ii), (iv): for κ < ∞ the growth condition in (ii) is vacuous
and V ≅ ℂ^{κ+1}; for κ = ∞, V contains the independent family {h_C}. ∎

**Corollary 3.2 (Berg–Meinardus [BM94]).** The Collatz conjecture holds iff
every solution h ∈ H(D) of (BM) has the form h = c₀ + c₁·z/(1−z) with
c₀, c₁ ∈ ℂ; equivalently iff dim_ℂ V = 2, i.e. V = span{1, z/(1−z)}.

*Proof.* If the conjecture holds, ℕ≥1 is a single component (every orb(n) ∋ 1,
so n ~ 1), so κ = 1 and by Prop. 3.1 every solution is
c₀·1 + c₁·z/(1−z); both functions do solve (BM) (constant on components,
bounded coefficients). If the conjecture fails, there is an exotic component C,
and h_C ∈ V; h_C ∉ span{1, z/(1−z)}, since h_C = c₀ + c₁ z/(1−z) would force,
comparing coefficients at some n ∈ C and at n = 1 ∉ C, both c₁ = 1 and c₁ = 0. ∎

So each exotic component (exotic cycle or divergent-orbit class) contributes
exactly one extra kernel dimension, with basis vector its indicator series.

## 4. The bi-Mahler exclusion

**Definition 4.1 (k-Mahler).** Let K be a field of characteristic 0 and k ≥ 2
an integer. F ∈ K[[z]] is *k-Mahler over K* if there exist d ≥ 0 and
p₀, …, p_d ∈ K(z), not all zero, with

    Σ_{i=0}^{d} p_i(z) · F(z^{k^i}) = 0.

Adamczewski–Bell use the a priori more liberal inhomogeneous polynomial form
[AB, eq. (1.3)]: A(z) + Σ_{i=0}^{n} P_i(z) F(z^{k^i}) = 0 with
A, P₀, …, P_n ∈ K[z] not all zero.

**Remark 4.2 (the two definitions agree).** Homogeneous ⟹ inhomogeneous: clear
denominators (take A = 0). Inhomogeneous ⟹ homogeneous: in a witness
A + Σ P_i F(z^{k^i}) = 0, some P_i is nonzero (else A(z) = 0 identically,
contradiction). If A = 0 we are done. Otherwise substitute z ↦ z^k to get
A(z^k) + Σ_i P_i(z^k) F(z^{k^{i+1}}) = 0 and cross-multiply:

    A(z^k)·[A(z) + Σ_i P_i(z) F(z^{k^i})] − A(z)·[A(z^k) + Σ_i P_i(z^k) F(z^{k^{i+1}})] = 0

is a homogeneous equation Σ_{i=0}^{n+1} Q_i(z) F(z^{k^i}) = 0 with
Q_{j+1} = A(z^k)P_{j+1}(z) − A(z)P_j(z^k). Taking j maximal with P_j ≠ 0 gives
Q_{j+1} = −A(z)P_j(z^k) ≠ 0. ∎

Note also that every rational F ∈ K(z) ∩ K[[z]] is k-Mahler for every k: if
F = P/Q then Q(z)P(z^k)·F(z) − P(z)Q(z^k)·F(z^k) = 0, and the two coefficients
are not both zero unless P = 0 (then F = 0 = 1·F). So "rational" is the minimal
possible intersection of Mahler classes, and the following theorem says it is
exactly the intersection.

**Theorem 4.3 (Adamczewski–Bell; Loxton–van der Poorten conjecture).** Quoting
[AB, Théorème/Theorem 1.1] verbatim: *"Let K be a field of characteristic zero
and let k and ℓ be two multiplicatively independent positive integers. Then a
power series F(z) ∈ K[[z]] is both k- and ℓ-Mahler if and only if it is a
rational function."* Here k, ℓ are multiplicatively independent iff there is no
pair of positive integers (n, m) with k^n = ℓ^m [AB, §1].

*Hypothesis check for our use.* K = ℚ has characteristic 0. k = 2, ℓ = 3 are
multiplicatively independent: 2^n is even and 3^m is odd for n, m ≥ 1, so
2^n = 3^m has no solution in positive integers. By Remark 4.2, "k-Mahler" in
Definition 4.1 implies k-Mahler in the sense of [AB, (1.3)], so Theorem 4.3
applies to Definition 4.1 as well.

**Lemma 4.4 (rational + finitely many coefficient values ⟹ eventually
periodic).** Let K be a field, F = Σ a_n z^n ∈ K[[z]] rational (QF = P for some
P, Q ∈ K[z], Q ≠ 0), with {a_n : n ≥ 0} finite of cardinality V. Then (a_n) is
eventually periodic: there are N ≥ 0 and p ≥ 1 with a_{n+p} = a_n for all
n ≥ N.

*Proof.* Write Q = z^v Q̃ with Q̃(0) ≠ 0. From z^v Q̃ F = P and F ∈ K[[z]] we get
z^v | P in K[z]; cancelling, Q̃F = P̃ with Q̃ = Σ_{i=0}^{d} q_i z^i, q₀ ≠ 0,
P̃ ∈ K[z]. Comparing coefficients of z^n for n > max(deg P̃, d−1):

    a_n = −q₀^{−1} Σ_{i=1}^{d} q_i a_{n−i},

so from some index N₀ on, a_n is a fixed function of the window
(a_{n−d}, …, a_{n−1}). (If d = 0 then F is a polynomial times q₀^{−1} and the
sequence is eventually 0; take p = 1.) There are at most V^d distinct windows,
so among the windows starting at N₀, N₀+1, …, N₀+V^d two coincide, say at s < t.
Since equal windows generate equal futures under the deterministic recurrence,
a_{n+(t−s)} = a_n for all n ≥ s. Take N = s, p = t − s. ∎

*Remark.* For integer (here 0/1) coefficients this is classically seen
analytically: by Fatou's lemma (1906) a rational power series with integer
coefficients can be written P/Q with P, Q ∈ ℤ[z] and Q(0) = 1, so the
reciprocal roots of Q are algebraic integers, stable under Galois; bounded
coefficients force them into the closed unit disk, their norms are then nonzero
rational integers of absolute value ≤ 1, hence all conjugates lie *on* the unit
circle, and Kronecker's theorem (1857) makes them roots of unity — i.e. all
poles of F are at roots of unity, which with boundedness yields eventual
periodicity by partial fractions. The pigeonhole proof above is shorter and
self-contained, so it is the one we rely on.

**Lemma 4.5 (AP extraction).** Let C ⊆ ℕ be infinite with eventually periodic
indicator sequence (period p ≥ 1 from index N on). Then C contains an infinite
arithmetic progression {n₀ + jp : j ≥ 0} with n₀ ≥ 1.

*Proof.* For n, n′ ≥ N with n ≡ n′ (mod p), iterating a_{·+p} = a_· gives
1_C(n) = 1_C(n′): membership beyond N depends only on the residue mod p. C is
infinite, so pick n₀ ∈ C with n₀ ≥ max(N, 1); then n₀ + jp ∈ C for all j ≥ 0. ∎

**Input: Monks's sufficiency theorem.** Monks works with the *accelerated* map
T̃ : ℤ⁺ → ℤ⁺, T̃(x) = x/2 for x even, T̃(x) = (3x+1)/2 for x odd. Two positive
integers x, y *merge* if their T̃-orbits intersect; a set S ⊆ ℤ⁺ is *sufficient*
if every positive integer merges with some element of S.

**Theorem (Monks [Mo06, Theorem 1.1]).** *Every arithmetic progression is
sufficient* — where an arithmetic progression is A + Bℕ = {A, A+B, A+2B, …}
with A, B nonnegative integers, B ≠ 0.

**Lemma 4.6 (transfer between T̃ and T).** For every x ∈ ℤ⁺,
orb_{T̃}(x) ⊆ orb_T(x).

*Proof.* If x is even, T̃(x) = T(x). If x is odd, T(x) = 3x+1 is even and
T²(x) = (3x+1)/2 = T̃(x). So each T̃-step is one or two T-steps; induct on the
number of steps. ∎

**Theorem 4.7 (bi-Mahler exclusion — main result).** Let C be an exotic
component of the Collatz graph (a weakly connected component of G on ℕ≥1 with
1 ∉ C). Then h_C = Σ_{n∈C} z^n does not satisfy both a 2-Mahler and a 3-Mahler
equation over ℚ(z). The same holds with ℚ replaced by any field of
characteristic zero.

*Proof.* Suppose h_C is both 2-Mahler and 3-Mahler over K (char K = 0).

(a) By Theorem 4.3 (hypotheses checked above), h_C is a rational function.

(b) The coefficients of h_C lie in {0, 1}, so by Lemma 4.4 the indicator
sequence of C is eventually periodic.

(c) C is infinite (Lemma 1.2(iii)), so by Lemma 4.5 it contains an infinite
arithmetic progression A = {n₀ + jp : j ≥ 0}, n₀ ≥ 1, p ≥ 1.

(d) By Monks's theorem applied to the progression A with the integer x = 1:
1 merges with some s ∈ A, i.e. there are k, l ≥ 0 with T̃^k(s) = T̃^l(1). Since
orb_{T̃}(1) = {1, 2} (T̃(1) = 2, T̃(2) = 1), we get T̃^k(s) ∈ {1, 2}, and by
Lemma 4.6, T̃^k(s) ∈ orb_T(s). Now s ∈ A ⊆ C and C is forward-closed
(Lemma 1.2(i)), so orb_T(s) ⊆ C; hence 1 ∈ C or 2 ∈ C — and in the latter case
1 = T(2) ∈ C as well.

(e) Either way 1 ∈ C, contradicting exoticness. ∎

Via Lemma 2.1, Theorem 4.7 says: **no extra kernel dimension of 𝔏 is
(2,3)-bi-Mahler.** The kernel elements that Collatz permits — 1 and z/(1−z) —
are rational, hence k-Mahler for every k; every kernel element beyond them
fails to live in both Mahler classes at once. This is the functional-equation
avatar of ×2×3 (base-2/base-3) disjointness.

**Corollary 4.8 (divergence certificates; the two-bases v2 statement).** Call
L ⊆ ℕ≥1 a *divergence certificate* if L ≠ ∅, T(L) ⊆ L, and 1 ∉ L. If
h_L = Σ_{n∈L} z^n is both 2-Mahler and 3-Mahler, then L is finite — and a
finite certificate contains a nontrivial cycle of T (which, by current
verification records, has all elements > 2^71 [Ba25] and is not an m-cycle for
m ≤ 91 [He23]). In particular, if the 3x+1 map has no nontrivial cycle, then no
bi-Mahler divergence certificate exists at all.

*Proof.* As in Theorem 4.7(a)–(b), L is eventually periodic. If L were
infinite, Lemma 4.5 gives an infinite AP inside L, and step (d) gives s ∈ L
whose T-orbit meets {1, 2}; T-invariance puts orb_T(s) ⊆ L, so L ∩ {1, 2} ≠ ∅
and then 1 ∈ L (as T(2) = 1) — contradiction. So L is finite. Iterating T from
any element of L stays in the finite set L, so by pigeonhole some element
repeats: L contains a cycle of T, which avoids 1, hence is nontrivial. ∎

(Unlike a component, a certificate need not be infinite — it can consist of an
exotic cycle alone — hence the cycle caveat, exactly as in the automatic-set
version of `two-bases.md`.)

**Corollary 4.9 (automatic and regular sets).** No exotic component C is both
2-automatic and 3-automatic; more generally, the indicator sequence of C is not
both 2-regular and 3-regular, and Corollary 4.8 likewise holds with
"k-automatic" or "k-regular" in place of "k-Mahler".

*Proof.* The chain k-automatic ⟹ k-regular ⟹ k-Mahler: a k-automatic sequence
is k-regular with finitely many values (Eilenberg kernel characterization;
[AS, Thm 16.1.5], quoted as [AB, Prop. 7.1(ii)]), and "Becker [Be94, Theorem 1]
showed that a k-regular power series is necessarily k-Mahler" [AB, §7.2]. Now
apply Theorem 4.7 / Corollary 4.8. ∎

Both containments are strict, so Theorem 4.7 strictly extends the Cobham-based
proposition of `two-bases.md`: n ↦ n is 2-regular but not 2-automatic
(automatic sequences take finitely many values), and Σ 2^n z^n = 1/(1−2z) is
k-Mahler for every k (rational) but k-regular for none (k-regular sequences
have polynomial growth [AB, Prop. 7.2]).

**Relation to the DFA searches.** The repo result "no regular divergence
certificate with ≤ 7 DFA states, base 2 LSB-first" (`experiments/dfacert`,
README) and the base-3 search (`experiments/dfacert3`) explore certificate
spaces that Corollary 4.9 proves disjoint except through eventually periodic
sets — and eventually periodic certificates are themselves excluded up to the
cycle caveat. A candidate found by the base-2 search can never be validated by
the base-3 search or vice versa; the two channels can only both succeed on a
nontrivial cycle. Theorem 4.7 extends this disjointness from the automaton
level to the far larger Mahler level: even certificate searches over 2-Mahler
and 3-Mahler *functional equations* (annihilator searches, as proposed in
CRACKS.md Crack 1, first move (b)) are provably disjoint channels.

## 5. Discussion: what one-sided rigidity would give, and what blocks it

Theorem 4.7 is two-sided: it needs a putative certificate to be visible in
*both* bases. The prize target is one-sided:

> **(R₂) Open target.** If F ∈ ℚ[[z]] is 2-Mahler, has coefficients in {0, 1},
> and its support S satisfies T(S) ⊆ S and 1 ∉ S, then F = 0.

(R₂) would kill every 2-automatic, 2-regular (0/1-valued), and 2-Mahler
divergence certificate at every state/degree count simultaneously — subsuming
the ≤7-state exhaustion and every future extension of it, since by
Corollary 4.9's chain each such certificate produces exactly such an F. Note
the target must be formulated for invariant *supports*, not for kernel
elements: a 2-automatic certificate L does yield an exotic 0/1 element of
ker 𝔏, namely the saturation

**Lemma 5.1 (saturation).** If L is a divergence certificate, then
L̂ := {n ≥ 1 : orb_T(n) ∩ L ≠ ∅} = ⋃_{j≥0} T^{−j}(L) is a nonempty union of
exotic components; hence h_{L̂} ∈ ker 𝔏 and h_{L̂} ∉ span{1, z/(1−z)}.

*Proof.* L̂ is a union of ~-classes: let n ∈ L̂, m ~ n. By Lemma 1.1 pick
q = T^i(n) ∈ orb(m), and w = T^j(n) ∈ L. Then T^{max(i,j)}(n) lies both in L
(T-invariance from w) and in orb(q) ⊆ orb(m), so m ∈ L̂. No component of L̂
contains 1: otherwise 1 ∈ L̂, so orb(1) = {1, 4, 2} meets L, and T-invariance
forces 1 ∈ L (from 4: T²(4) = 1; from 2: T(2) = 1), contradiction. h_{L̂} is a
0/1 assignment constant on components, so h_{L̂} ∈ ker 𝔏 by Lemma 2.1, and it
is exotic as in Corollary 3.2. ∎

— but saturation is an infinite union: each T^{−j}(L) of a 2-automatic L is
again 2-automatic (Büchi-definability of affine maps), yet the union over all j
is not uniform, and no argument is known that L̂ (or any exotic component)
inherits 2-Mahlerness from L. So proving "no exotic *component* is 2-Mahler"
would not, by itself, retire the DFA program; (R₂) is the statement that would.

**The obstructions, stated precisely.**

1. *Λ₃ is not a ring map.* Λ₃ = Λ_{3,2} is ℚ-linear but kills the unit:
   Λ₃(1) = 0, so it is not a homomorphism of any function ring, let alone a
   field endomorphism; it is a section operator, obeying the twisted Leibniz /
   splitting identities Λ_{3,r}(M₃(f)·g) = f·Λ_{3,r}(g) and
   g = Σ_{r=0}^{2} z^r · M₃(Λ_{3,r} g). By contrast, all existing Mahler–Galois
   machinery — difference-Galois theory for the endomorphism
   σ_k : f(z) ↦ f(z^k) of a Mahler field such as ⋃_j K((z^{1/k^j})) ([DHR],
   [Ro25]), σ-Galois theory for Mahler systems and their independence ([ADHW],
   [SS]) — is built over an injective endomorphism of the base field. The
   operator 𝔏 = 1 − M₂ − z·M₂·Λ₃ generates with ℚ(z) a crossed algebra
   ℚ(z)⟨M₂, Λ₃⟩ containing a non-injective, non-multiplicative operator; no
   Picard–Vessiot category currently contains it.

2. *Finiteness fails a priori.* Picard–Vessiot / σ-Galois theory attaches
   groups to finite-dimensional solution modules. By Proposition 3.1,
   dim ker 𝔏 (in H(D)) = 1 + #components of G on ℕ≥1, and the finiteness of
   the number of components is itself an open Collatz-adjacent statement: it
   implies finiteness of the number of cycles, currently known only for
   m-cycles with m ≤ 91 [He23]. A Galois-theoretic route to (R₂) must therefore
   either work "one certificate at a time" — e.g. close the single 2-Mahler
   equation satisfied by a putative certificate into a finite M₂-system; the
   crack brief's checked closure observation (BM rearranges to an inhomogeneous
   order-1 2-Mahler equation for M₃h with μ_{3^k}-twists, the twist tower being
   the technical wall) points there, but is not re-derived in this note — or
   develop descent for infinite-dimensional kernels. Neither exists today.

3. *One-sidedness must use the dynamics.* The bi-Mahler input (Theorem 4.3) is
   sharp as pure Mahler theory: single-base Mahler classes are enormous (they
   contain all k-automatic and k-regular series), so (R₂) cannot follow from
   Mahler rigidity alone — the mixed operator 𝔏, i.e. the Collatz dynamics
   itself, must enter. That is what makes (R₂) a theorem-shaped target rather
   than a corollary: it sits one step past [AB]/[SS]/[ADHW] in an algebra those
   theories do not yet contain.

Calibration: like the two-bases note, everything here constrains disproofs and
certificate syntax. By the Conway / Kurtz–Simon undecidability ceiling, no
extension of this method alone decides the conjecture; its value is that it
retires certificate classes wholesale and, in (R₂), names a concrete
rigidity statement whose proof would subsume the entire base-2 automatic
program in one stroke.

   (in particular the ω-average is ω-free, has no pole at 0, and all
   m ≢ 0 (mod 3) coefficients vanish);
2. the residual equals (𝔏h)(z³) for 𝔏 = 1 − M₂ − z·M₂·Λ_{3,2};
3. the relation systems {a_{2k}−a_k, a_{2k+1}−a_{3k+2}} and {a_n − a_{T(n)}}
   generate the same linear span (40 relations each way, window-honest
   membership tests);
4. a random rational assignment constant on the brute-force-computed components
